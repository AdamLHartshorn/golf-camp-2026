# Shenanigans Notes

Shenanigans is a point-based side game system used mostly outside official morning rounds.

Current important feedback:
- Settlement should calculate from all Shenanigans point events.
- Dollar-per-point multiplier should be configurable.
- Wagers need custom point amounts.
- Future architecture should support initiated game sessions for groups of four.
- Game sessions should start each player with 5 points.
- Forced events/random challenges are a long-term idea, not immediate scope.
